# Запуск проекта Questionnaire-Builder-App

Этот проект разработан для компании osSystem. Чтобы запустить проект, выполните следующие шаги:

1. Склонируйте репозиторий на свой компьютер, используя следующую команду в командной строке:

```
git clone https://github.com/VitaliiBarabash1990/Questionnaire-Builder-App
```

2. Перейдіть в папку з проектом:

```
   cd Questionnaire-Builder-App
```

3. Встановите заледності, використовуючи менеджер пакетів `npm`:

```
   npm install
```

4. Запустите проект, используя команду:

```
   npm start dev
```

# Список хуків

| Хук           | Опис                                                                                                                                        |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| useCallback   | Створює мемоизированный колбек, що допомогає уникнути повторних обчислень і покращити продуктивність                                        |
| useMemo       | Створює мемоізованне значеня, що доаомагає кешувати результати обчислень и покращити продуктивність.                                        |
| useState      | Дозволяє створювати змінні стани та оновлювати їх при зміні введення користувача або інших подій.                                           |
| useEffect     | Дозволяє виконати побічний ефект при кожному рендері компонента, що дозволяє взаємодіяти із зовнішнім світом та оновлювати стан компонента. |
| useSelector   | Дозволяє отримувати дані зі сховища Redux у компоненті.                                                                                     |
| useDispatch   | Дозволяє надсилати дії Redux з компонента.                                                                                                  |
| useNavigate   | Дозволяє програмно переходити на інші сторінки у програмі з використанням React Router.                                                     |
| useMediaQuery | Дозволяє отримувати інформацію про розміри екрана та стан медіа-запитів.                                                                    |
